Memoy game using jQuery.

Rules: click on 2 photos. If the photos are a matching pair, they will stay face up, else third click will flip them over and open a new photo.
try to get all the pairs face up in a few moves as possible.

Note: The board is created dynamically hence, you can easily change the number of the board size
by changing the 'rows' and 'cols'. Then add more photos to 'Img' array.
